# 🧠 AI Health Prediction System

An end-to-end Machine Learning project that predicts:

* 🩺 **Diabetes Risk (Classification)**
* 💰 **Insurance Cost (Regression)**

This project demonstrates the complete ML pipeline — from data preprocessing to model deployment using Streamlit.

---

## 🚀 Features

* 🔍 Multiple ML models trained & compared
* 🧠 Best model selection based on performance
* ⚙️ Hyperparameter tuning
* 📊 Evaluation using accuracy, precision, recall, F1-score
* 🖥️ Interactive Streamlit UI
* ⚡ Real-time predictions

---

## 🧠 Models Used

### Classification (Diabetes Prediction)

* Logistic Regression
* Decision Tree
* Random Forest ✅ *(Best Model)*
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* Gradient Boosting
* Naive Bayes

### Regression (Insurance Prediction)

* models = 
    "Linear Regression"
    "Polynomial"
    "Ridge"
    "Lasso"
    "ElasticNet"
    "Decision Tree"
    "Random Forest"
    "Gradient Boosting"
    "SVR"
    "KNN"

---

## 📊 Model Performance

After comparing multiple models:

* 🏆 **Random Forest performed best** for diabetes prediction
* Achieved strong accuracy and balanced performance
* More stable than probabilistic models like Naive Bayes

## ▶️ Run Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/ranjithjunnu09/Classification_Regression_project.git
cd Classification_Regression_project
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the app

```bash
streamlit run app.py
```
